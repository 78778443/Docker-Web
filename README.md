# Docker-Web
1. 目录结构：
  .
  ├── config.php  配置文件
  ├── css         css文件目录
  ├── docker.php  后台入口文件
  ├── fonts
  ├── index.php   前台入口文件
  ├── js          js文件目录
  ├── libs        类文件
  └── static      html文件

2. 功能简介：
  支持批量删除镜像和容器;
  支持镜像的创建;
  支持镜像标签;
  支持镜像的history和inspect查看;
  支持容器的常规操作，比如重启，暂停等;
  支持容器的重命名、进程查看、容器内文件系统的改变以及inspect的查看;
